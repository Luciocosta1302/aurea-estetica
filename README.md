# Áurea Estética Avançada — Landing Page

Landing page fictícia de clínica de estética, construída em **HTML, CSS e JavaScript puros** (sem frameworks). Projeto de portfólio.

**Ver online:** https://luciocosta1302.github.io/aurea-estetica/

## Destaques

- Paleta nude/rosé + terracota com tipografia serifada (Fraunces) e formas orgânicas em cápsula
- Abas de especialidades (Harmonização Facial, Tratamentos Faciais e Corporais)
- Comparador **antes/depois** interativo (arraste)
- Botão flutuante de **WhatsApp** e CTAs de agendamento
- Responsiva, com `prefers-reduced-motion` e fallback local para imagens
- Deploy automático no GitHub Pages via Actions

## Personalização

- **WhatsApp:** troque `5511999999999` nos links `wa.me` do `index.html`
- **Cores e fontes:** variáveis CSS em `assets/styles.css` (`:root`)
